# AdReply Chrome Extension

## Overview

AdReply is an intelligent Facebook comment advertising assistance tool designed for small business owners and marketers. It provides contextual template suggestions for engaging with potential customers in Facebook groups and posts.

## Features

- **Category-Based Templates**: 400+ pre-built templates organized into 20+ business categories
- **Smart Template Matching**: AI-powered keyword matching for relevant suggestions
- **Anti-Spam Protection**: 24-hour cooldown system prevents repetitive messaging
- **Category Preferences**: Set preferred business categories for personalized suggestions
- **Professional Templates**: All templates comply with Facebook community standards
- **Usage Analytics**: Track template performance and usage statistics

## Installation

1. Download the extension files
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked" and select the extension folder
5. The AdReply icon will appear in your Chrome toolbar

## Usage

1. Navigate to Facebook.com
2. Click the AdReply icon to open the side panel
3. Set your preferred business category in settings
4. Browse Facebook groups and posts
5. Use suggested templates to engage with potential customers
6. Templates automatically include your promotional URL

## Privacy & Security

- No personal data is collected or transmitted
- All processing happens locally in your browser
- Templates and usage data are stored locally only
- Complies with Facebook's community standards and advertising policies
