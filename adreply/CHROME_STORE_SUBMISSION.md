# Chrome Web Store Submission Package

## Package Contents
This folder contains the complete AdReply Chrome Extension ready for Chrome Web Store submission.

## Files Included
- `manifest.json` - Extension manifest (Manifest V3)
- `README.md` - Extension documentation
- `PRIVACY_POLICY.md` - Privacy policy for Chrome Web Store
- `assets/` - Extension icons and resources
- `scripts/` - Background and content scripts
- `storage/` - Data management and storage modules
- `ui/` - User interface files and modules

## Key Features
- **Category-Based Templates**: 400+ professional templates across 20+ business categories
- **Smart Matching**: Intelligent keyword-based template suggestions
- **Anti-Spam Protection**: 24-hour cooldown system per template per group
- **Facebook Compliance**: All templates meet Facebook community standards
- **Privacy-First**: No data collection, all processing happens locally

## Technical Specifications
- **Manifest Version**: 3 (latest Chrome extension standard)
- **Permissions**: Minimal required permissions only
- **Host Permissions**: Limited to Facebook.com only
- **Storage**: Local Chrome storage API only
- **Privacy**: No external data transmission
